# Instagram-like Social Media App

## Overview
This app mirrors Instagram's functionality, allowing users to create, edit, delete, save, and manage posts, as well as edit their profiles.

## 1.  Tech Stack Used:

   - **React.js**
   - **Appwrite**
   - **React Query**
   - **TypeScript**
   - **Shadcn**
   - **Tailwind CSS**
